Um jogo da memória divertido feito com **HTML**, **CSS** e **JavaScript**.  
O objetivo é encontrar todos os pares de cartas no menor tempo possível.

---

## 🚀 Como jogar
1. Abra o arquivo `index.html` no navegador.
2. Clique nas cartas para revelar as imagens.
3. Encontre os pares corretos até completar o tabuleiro.
4. Ganhe ao encontrar todos os pares!

---

## 📂 Estrutura do projeto 
 index.html   # Página principal
 style.css    # Estilização visual
 script.js    # Lógica do jogo
 .gitignore   # Arquivos ignorados pelo Git
 README.md    # Documentação do projeto

# Funcionalidades
-Animação ao virar cartas  
-Contagem de pares encontrados  
-Reinício automático ao finalizar  
-Fácil de personalizar com novas imagens
